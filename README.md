# NoteIT

- 단일 주제의 여러 핵심 키워드를 스스로 정리하고 반복 학습하는 애플리케이션

## 주요 기능

- 단어장에 주제와 세부 키워드, 이미지를 포함한 상세 설명
- 카테고리, 주제 북마크
- 단답식 퀴즈와 점수 표시
- 키워드별 틀린 횟수 표시
- 점수 통계 표시

## 부가 기능

- 페이지 프린트
- 프로필 관리
- 배경 음악
- 다크 모드
- 중요도, 완료기한을 포함한 투두리스트

## 개발 도구

- React
- Node.js
- MySQL
- Express
- Sequelize
